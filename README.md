# Vibrant Foodie AI

Aplicación de nutrición y bienestar con recetas inteligentes, retos, seguimiento de hábitos y contenido de administración.

## Tecnologías

- React 18 + Vite 5 + TypeScript 5
- Tailwind CSS
- Lovable Cloud

## Notas de compilación

La importación de `AuthContext` en `src/App.tsx` usa una ruta relativa (`./contexts/AuthContext`) para garantizar la compatibilidad con el empaquetador de producción.
